# Privacy Policy for HOME 5G L13 電波モニター

This Privacy Policy applies to the HOME 5G L13 電波モニター Android application.

The application provides network and signal monitoring information from a locally configured router interface and device environment. It does not collect or store personally identifiable information.

## Data Collection

The app collects and displays the following technical data:

- Base station information (cell tower data)
- Signal strength metrics (RSRP / RSSI)
- Network status information retrieved from a user-configured local router interface (default gateway such as http://192.168.0.1).

The app does not collect:
- GPS location data
- Contacts, photos, or files
- Device identifiers such as IMEI

## Network Communication

The application communicates with a user-configured local router management interface.

By default, this may be:
- http://192.168.0.1 or other gateway addresses depending on the network environment

However, users may configure the router IP address according to their network setup.

Authentication (username/password) may be required depending on router settings.

No personal data collected by the application is transmitted to external servers other than those used by Google AdMob services.

Network communication is limited to:
- Local router interface access
- Google AdMob services operated by Google

## Third-Party Services

This application uses Google AdMob to display advertisements.

AdMob is operated by Google and may collect and process data such as:
- Device advertising ID
- IP address
- Approximate location (if enabled by the system)
- Usage data for ad personalization

This data is handled by Google in accordance with its Privacy Policy.

## Data Storage

No personal user data is stored or transmitted by this application.

## Analytics

This application does not use analytics services such as Firebase Analytics.
