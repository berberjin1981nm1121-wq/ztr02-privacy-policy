# 電波見太郎 for L13 プライバシーポリシー

本プライバシーポリシーは、Androidアプリ「電波見太郎 for L13」に適用されます。

本アプリは、ローカルに設定されたルーターインターフェースおよび端末環境から、ネットワーク情報や電波情報を表示するためのアプリです。本アプリ自体は、個人を特定できる情報を収集または保存しません。

## データ収集について

本アプリは以下の技術情報を取得・表示します。

- 基地局情報
- 電波強度情報（RSRP / SINR / RSRQ / SNR /RSSI）
- ユーザーが設定したローカルルーターインターフェース（例: http://192.168.0.1）から取得したネットワーク情報

本アプリは以下を取得しません。

- GPS位置情報
- 連絡先、写真、ファイル
- IMEIなどの端末識別情報

## ネットワーク通信について

本アプリは、ユーザーが設定したローカルルーター管理インターフェースと通信します。

デフォルトでは以下のようなゲートウェイアドレスを使用する場合があります。

- http://192.168.0.1 など、ネットワーク環境に応じたローカルアドレス

ユーザーはネットワーク環境に応じてルーターIPアドレスを変更できます。

また、ルーター設定に応じて認証（ユーザー名・パスワード）が必要になる場合があります。

本アプリは、Google AdMob や通信速度測定機能などの第三者サービスに必要な場合を除き、個人情報を意図的に外部サーバーへ送信しません。

通信には以下が含まれる場合があります。

- ローカルルーターインターフェースへのアクセス
- 通信速度測定機能のための外部サーバー通信（Cloudflare Speed Test など）
- Google AdMob による広告通信

通信速度測定機能は、ダウンロード速度、アップロード速度、遅延（レイテンシ）などのネットワーク性能測定のみに使用されます。

## 第三者サービスについて

本アプリは広告表示のために Google AdMob を使用しています。

Google AdMob は以下の情報を収集・処理する場合があります。

- 広告ID
- IPアドレス
- おおよその位置情報（端末設定で有効な場合）
- 広告最適化のための利用情報

これらのデータは Google のプライバシーポリシーに従って処理されます。

## データ保存について

本アプリは個人情報を保存しません。

## 解析ツールについて

本アプリは Firebase Analytics などの解析サービスを使用していません。

## お問い合わせ

サポートについては GitHub Issues をご利用ください。

https://github.com/berberjin1981nm1121-wq/ztr02-privacy-policy/issues

# Privacy Policy for 電波見太郎 for L13

This Privacy Policy applies to the 電波見太郎 for L13 Android application.

The application provides network and signal monitoring information from a locally configured router interface and device environment. It does not collect or store personally identifiable information.

## Data Collection

The app collects and displays the following technical data:

- Base station information (cell tower data)
- Signal strength metrics（RSRP / SINR / RSRQ / SNR /RSSI）
- Network status information retrieved from a user-configured local router interface (default gateway such as http://192.168.0.1).

The app does not collect:

- GPS location data
- Contacts, photos, or files
- Device identifiers such as IMEI

## Network Communication

The application communicates with a user-configured local router management interface.

By default, this may include:

- http://192.168.0.1 or other gateway addresses depending on the network environment

However, users may configure the router IP address according to their network setup.

Authentication (username/password) may be required depending on router settings.

The application does not intentionally collect or transmit personal user data to external servers, except as required for third-party services such as Google AdMob and optional network speed testing functionality.

Network communication may include:

- Local router interface access
- External network servers used for optional network speed test functionality (such as Cloudflare speed test services)
- Google AdMob services operated by Google

Speed test communications are used solely to measure network performance such as download speed, upload speed, and latency.

## Third-Party Services

This application uses Google AdMob to display advertisements.

AdMob is operated by Google and may collect and process data such as:

- Device advertising ID
- IP address
- Approximate location (if enabled by the system)
- Usage data for ad personalization

This data is handled by Google in accordance with its Privacy Policy.

## Data Storage

The application does not store personal user data.

## Analytics

This application does not use analytics services such as Firebase Analytics.
